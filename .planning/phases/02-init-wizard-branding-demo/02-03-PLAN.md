  <action>
Create directory `assets/` if it does not exist, then create `assets/demo.tape` -- a VHS tape file for reproducible terminal GIF recording (per D-34).

The tape file should simulate a design-review run with branded output. Since a real `/design-review` takes 8-10 minutes and requires a live dev server, the tape should show pre-crafted output that demonstrates the branded format (per Research open question 1).

**VHS tape configuration (at the top of the file):**
- Add a comment: `# SpSk Demo Recording`
- Add a comment: `# After recording: gifsicle --lossy=80 --optimize=3 --colors 64 -o assets/demo.gif assets/demo-raw.gif`
- Set Output to assets/demo-raw.gif
- Set Shell bash
- Set FontSize 14
- Set FontFamily "Geist Mono"
- Set Width 1200
- Set Height 800
- Set Theme "Catppuccin Mocha"
- Set WindowBar Colorful
- Set TypingSpeed 40ms
- Set Padding 20

**Recording flow (target: 30 seconds of GIF, per D-33):**

1. Show the command being typed (2s): Type the claude design-review command, Enter, Sleep 1s

2. Show the signature line appearing (1s): Type the SpSk signature line with version, specialist count, tier

3. Show 3-4 specialist boxes scrolling by with scores (8-10s): Use Type commands to output pre-formatted specialist sections in branded format. Show Typography, Color, Layout, and Intent specialists (representative subset). Each with a single-line Unicode box header, score bar, and 2-3 findings using the symbol vocabulary (checkmark for pass, cross for fail).

4. Show a summary line like "... 4 more specialists evaluated ..." (1s)

5. Show final verdict box with overall score and SHIP verdict (3s): A final single-line box with the weighted score bar, SHIP verdict, and 2-3 key findings.

6. Show footer (1s): Type the github.com/felipemachado/spsk footer line, Sleep 2s

**Important notes:**
- Use Sleep between sections for readability
- The specialist output should use the exact branding from shared/output.md (single-line boxes with ┌─┐│└─┘, score bars with ████░░ on /10 display scale, symbols ✓ ✗ ◆ ○ ⚡ ⚠)
- Total typing + sleep should result in approximately 25-30 seconds of recording
- Output to assets/demo-raw.gif (gifsicle compresses to assets/demo.gif)
  </action>
