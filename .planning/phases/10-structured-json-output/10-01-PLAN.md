Standard specialist schema (font, color, layout, icons, motion, copy, code-a11y):
```json
{
  "specialist": "typography",
  "score": 3,
  "findings": [
    {
      "element": "h1.hero-title",
      "issue": "Hero uses Playfair Display",
      "recommendation": "Replace with Instrument Serif"
    }
  ],
  "summary": "One-line summary of evaluation"
}
```

Intent specialist schema (multi-score variant):
```json
{
  "specialist": "intent",
  "scores": {
    "intent_match": 3,
    "originality": 2,
    "ux_flow": 3
  },
  "findings": [
    {
      "dimension": "originality",
      "element": "hero section",
      "issue": "Three-column icon grid is common AI pattern",
      "recommendation": "Use asymmetric layout"
    }
  ],
  "summary": "One-line summary"
}
```

Boss synthesizer schema:
