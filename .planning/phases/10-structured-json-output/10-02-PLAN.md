Functions in parse-review-output.sh:
- extract_verdict(output) -> "SHIP" | "CONDITIONAL" | "BLOCK"
- extract_overall(output) -> float (e.g., "2.65")
- extract_score(dimension, output) -> float (e.g., "3")
- float_in_range(value, min, max) -> exit 0 (true) or 1 (false)
- verdict_matches(actual, expected) -> exit 0 (true) or 1 (false)

    **Add these new internal helper functions** (above the existing functions):

    1. `extract_json_block(output, tag)` -- Extracts raw JSON between `<tag>` and `</tag>` XML wrappers.
       ```bash
       extract_json_block() {
         local output="$1"
         local tag="$2"
         echo "$output" | sed -n "/<${tag}>/,/<\/${tag}>/p" | sed '1d;$d'
       }
       ```

    2. `extract_verdict_json(output)` -- Extracts verdict from `<boss_output>` JSON using jq.
       ```bash
       extract_verdict_json() {
         local output="$1"
         local json
         json=$(extract_json_block "$output" "boss_output")
         if [ -n "$json" ] && echo "$json" | jq -e '.verdict' &>/dev/null; then
           echo "$json" | jq -r '.verdict' | sed 's/CONDITIONAL SHIP/CONDITIONAL/'
         fi
       }
       ```

    3. `extract_overall_json(output)` -- Extracts weighted_score from `<boss_output>` JSON.
       ```bash
       extract_overall_json() {
         local output="$1"
         local json
         json=$(extract_json_block "$output" "boss_output")
         if [ -n "$json" ] && echo "$json" | jq -e '.weighted_score' &>/dev/null; then
           echo "$json" | jq -r '.weighted_score'
         fi
       }
       ```

    4. `extract_score_json(dimension, output)` -- Extracts per-specialist score from `<boss_output>` JSON.
       ```bash
       extract_score_json() {
         local dimension="$1"
         local output="$2"
         local json
         json=$(extract_json_block "$output" "boss_output")
         if [ -n "$json" ] && echo "$json" | jq -e ".scores.${dimension}" &>/dev/null; then
           echo "$json" | jq -r ".scores.${dimension}"
         fi
       }
       ```

    **Update the existing public functions** to try JSON first, then fall back to regex:

    - `extract_verdict()`: Try `extract_verdict_json "$output"` first. If result is non-empty, echo it and return. Otherwise, fall through to existing regex logic (keep current implementation as-is).

    - `extract_overall()`: Try `extract_overall_json "$output"` first. If result is non-empty, echo it and return. Otherwise, fall through to existing regex logic.

    - `extract_score()`: Try `extract_score_json "$dimension" "$output"` first. If result is non-empty, echo it and return. Otherwise, fall through to existing regex logic (dimension-to-label mapping + grep).

    **Do NOT change:**
